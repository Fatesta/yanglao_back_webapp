## 智慧养老平台后台Web前端项目


源码目录:
```
.
├── package.json
├── src                 # 源码
│   ├── components      # 公共/抽象组件  
│   ├── config          # 配置相关
│   ├── pages           # 业务页面
│   └── index.js        # SPA主入口
│   └── pages.js        # 页面配置
│   └── index.html      # html模板
│   └── index.css       # 基本的全局的样式
└── webpack.config.js
```
